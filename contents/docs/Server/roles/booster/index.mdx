## What is it ? 

Booster roles are unique roles that are given to members which have boosted your server. 
These roles can be fully customized by the members themselves.

## Getting Started

Gato does have a command base for setting up a base role. Base role will create new booster roles below the base role.
If no base role is setup, the new role will be created at the very bottom of the server.

<Note type="warning">
    For management reasons, it better to setup a base role. 
</Note>

- How to setup a base role : 

    <Tabs defaultValue="syntax" className="pt-3 pb-0.7">
    <TabsList>
        <TabsTrigger value="syntax">Syntaxe</TabsTrigger>
        <TabsTrigger value="example">Exemple</TabsTrigger>
      </TabsList>
      <TabsContent value="syntax">
        ```plaintext
        $boosterrole base set <role>
        ```
      </TabsContent>
      <TabsContent value="example">
        ```plaintext
        $boosterrole base set @BoosterRoles
        ```
      </TabsContent>
    </Tabs>

- How to create a boosterrole : 

    <Tabs defaultValue="syntax" className="pt-3 pb-0.7">
    <TabsList>
        <TabsTrigger value="syntax">Syntaxe</TabsTrigger>
        <TabsTrigger value="example">Exemple</TabsTrigger>
      </TabsList>
      <TabsContent value="syntax">
        ```plaintext
        $boosterrole create <role name>, <color> 
        ```
      </TabsContent>
      <TabsContent value="example">
        ```plaintext
        $boosterrole create Genius, red
        ```
      </TabsContent>
    </Tabs>

    <Note type="warning">
        Color can be an HEX color. 
    </Note>

## Booster Awards 

Grant a role whenever a member of your server boost it. You can use 
`booster award` commannd to set it up. 

<Tabs defaultValue="syntax" className="pt-2 pb-1">
                    <TabsList>
                    <TabsTrigger value="syntax">Syntaxe</TabsTrigger>
                    <TabsTrigger value="example">Exemple</TabsTrigger>
                    </TabsList>
                    <TabsContent value="syntax">
                    ```plaintext
                    $booster award add <roles>
                    ```
                    </TabsContent>
                    <TabsContent value="example">
                    ```plaintext
                    $booster award add rich
                    ```
                    </TabsContent>
                </Tabs>

To remove the booster role, use the command `booster award remove`. If you want to see them, do `booster award list`
